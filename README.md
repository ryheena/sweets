# Sweets
Next Web builder toolkit

## Conception

* Sweets
* No ORM
* RESTful API

## Features

* Modular
* Lightweight
* Auto api routing generation
* Database independent
* Authentication
* Focused on RESTful api creation
* API testing UI
* Client [lib](http://github.com/swts/swts) (but you can use your own)
